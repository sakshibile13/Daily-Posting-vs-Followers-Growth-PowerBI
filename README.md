##  Project Objective
The objective of this project is to analyze how daily posting frequency and different content types impact follower growth and engagement. The dashboard helps content creators and social media managers identify optimal posting strategies to maximize follower growth.

---

##  Dataset Used
- Social media performance dataset containing:
  - Date
  - Posts per day
  - Content type (Image, Video, Text, Carousel, Live)
  - Likes
  - Follower change
  - Viral spikes
- Dataset type: Simulated / anonymized social media data  
- Format: CSV / Excel

---

##  Business Questions / KPIs
- How does daily posting frequency affect follower growth?
- Which content type contributes the most to follower increase?
- What is the follower growth trend over time?
- How are posts distributed across different content types?
- Which content type delivers the highest engagement and growth?

---

##  Process / Methodology
1. Imported data into Power BI
2. Cleaned and transformed data using Power Query
3. Created DAX measures for:
   - Total follower growth
   - Posts per day
   - Viral spike count
4. Analyzed time-based follower growth trends
5. Compared performance across different content types
6. Designed interactive visuals and slicers for dynamic analysis

---

##  Dashboard Overview
The dashboard includes:
- KPI cards for viral spikes, total followers gained, and total posts
- Follower and posting trend over time
- Content type-wise engagement and growth comparison
- Follower growth contribution by content type
- Post distribution by content type
- Interactive slicers for content type selection

##  Project Highlights
- Image and Video content contribute the highest follower growth
- Consistent daily posting shows a strong positive impact on follower increase
- Carousel and Live posts have lower contribution but higher niche engagement
- Time-based analysis reveals steady follower growth with periodic viral spikes
- Dashboard provides actionable insights for content strategy optimization

---

##  Final Conclusion
This project demonstrates how posting frequency and content strategy directly influence social media growth. By analyzing content performance and follower trends, creators can optimize posting schedules and focus on high-impact content types to achieve sustainable follower growth.
